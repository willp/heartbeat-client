# Security Policy

## Supported Versions

The latest released version on PyPI is supported for security fixes.
Older versions may receive fixes at maintainer discretion.

## Reporting a Vulnerability

Please report security issues privately to `security@TenthLight.com`.

When reporting, include:

- A clear description of the issue and impact
- Reproduction steps or proof of concept
- Affected versions
- Any suggested mitigations

You can expect an initial response within 5 business days.

Please do not open a public issue for suspected vulnerabilities until we have
had a chance to investigate and prepare a fix.
